# Resume Website

A responsive resume website built using **HTML5** and **CSS3**. This project showcases a professional resume with a modern two-column layout, highlighting personal information, technical skills, education, experience, and projects.

## Features

- Responsive resume layout
- Semantic HTML5 structure
- Two-column design
- Profile image with circular border
- Contact information section
- Technical skills section
- Certifications section
- Coursework section
- Education timeline
- Experience section
- Projects showcase
- Custom gold diamond (◆) bullets in the sidebar
- Custom gold dash (–) bullets in the main content
- Professional typography using Montserrat
- Modern color palette

## Technologies Used
